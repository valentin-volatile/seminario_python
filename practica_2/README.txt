# Ejercicio 10: Simulación de partidas y ranking en shooter online

## Instalación de dependencias
Para instalar las dependencias especificadas en ´requirements.txt´ es necesario ejecutar este comando en la terminal:
´pip install -r requirements.txt´

Esto instalará Jupyter Notebook, el cual es necesario para ejecutar los archivos con extensión .ipynb.
Jupyter Notebook es una aplicación web que permite ejecutar y visualizar código en diferentes lenguajes de programación, en este caso Python.

## Ejecución del proyecto
Una vez instaladas las dependencias, ejecutamos el comando ´jupyter notebook´.
Esto nos abrirá una página en nuestro navegador donde podremos acceder y abrir los notebooks del proyecto.
Navegamos hacia la carpeta "notebooks", y abrimos el archivo 'Ej10.ipynb'
Para ver las funciones utilizadas podemos ir a la carpeta "src" y abrir el archivo "funciones.py".

Aclaración: el proyecto fue creado y ejecutado con la versión de escritorio de Jupyter Notebook (instalada de la forma anteriormente mencionada), no usando VSCode.
